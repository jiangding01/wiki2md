# 隐私说明

**一句话：你的数据不出你的浏览器。**

## 数据处理

- 所有提取、转换、打包都在你的浏览器本地完成，摘墨没有任何服务器，不向任何外部服务发送数据。
- 摘墨发起的网络请求仅限于**你正在导出的站点本身**：拉取该站点的评论（Confluence REST）、
  图片附件、子页面内容——这些请求携带的是你浏览器里已有的登录态，等同于你自己在页面上的访问。
- 导出历史与设置存储在浏览器本地（`chrome.storage`），可随时在设置页清空；
  「记录导出历史」可整体关闭。

## 权限说明

| 权限 | 用途 |
| --- | --- |
| `activeTab` + `scripting` | 你主动点击导出时，向**当前标签页**注入提取脚本。安装时不索取"读取所有网站"的常驻权限 |
| `storage` | 保存设置与导出历史（仅本机 + 浏览器自带同步） |
| `contextMenus` | 右键菜单「导出为 Markdown」 |
| `clipboardWrite` | 「复制 Markdown」按钮 |
| `tabs` + `<all_urls>`（**可选**，默认未开启） | 仅「批量导出多标签页」功能需要，使用时弹窗申请，可随时在 `chrome://extensions` 撤销 |
